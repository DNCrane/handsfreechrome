Hands Free for Chrome
A Chrome extension to browse the web using only your voice, free to download and use. A mic is required.

Install for Chrome

You're not using Google Chrome! Once you're in Chrome, come back to download Hands Free here, or get it from the Chrome Web Store.

Quick Start Guide
After installing Hands Free, restart Chrome, then open any webpage and hit Ctrl + Space. Give the input window permission to use your mic.
Then just say help.

Are you using a Mac?
In order to effectively use Hands Free on a Mac, first go to System Preferences, then to Sound, then to Input. If there's a little checkbox option that says "Use ambient noise reduction," make sure that it is not checked.

Hands Free can scroll around, click things, navigate to particular webpages, go back or forward in your browsing history, enter text into forms, check and uncheck checkboxes, zoom in and out, open and close tabs, refresh the page, and more. This is an initial release, and many improvements and additional features will soon follow.

For the average user, Hands Free permits easy browsing while eating or multitasking, finally bringing an end to the dark era in which a comfortable individual would have to physically move in order to click, scroll, or type.

For the handicapped user, Hands Free will immensely facilitate access to information and communication via the world wide web.

This extension is for the Chrome web browser, which can be downloaded for free from the Google Chrome website.
